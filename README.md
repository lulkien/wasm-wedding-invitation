# рџ’Ќ Wedding Invitation

A personalised, fullstack wedding invitation web app built with [Dioxus 0.7](https://dioxuslabs.com/learn/0.7). Each guest receives a unique link that greets them by name and lets them RSVP with their shuttle departure preference.

---

## Features

- **Personalised invitations** вЂ” each guest link (`/invitation/:uid`) shows a custom greeting
- **Live countdown** вЂ” ticks down to the ceremony in days, hours, minutes, seconds
- **RSVP** вЂ” guests select a shuttle departure point; the choice is persisted to a SurrealDB database
- **Zalo group links** вЂ” QR codes and links appear based on the chosen shuttle stop
- **Fully static config** вЂ” ceremony details, venue, and couple names live in a single JSON file; no code changes needed
- **SCSS pipeline** вЂ” styles are authored in SCSS and compiled to CSS at build time via `build.rs`

## Configuration

Edit `config/wedding.json` to customise all ceremony details вЂ” no code changes required.

```json
{
  "ceremony": {
    "date_utc": "2026-03-28T06:30:00Z",
    "date_display": "Saturday, March 28, 2026",
    "reception_time": "8:30 AM",
    "ceremony_time": "1:30 PM"
  },
  "venue": {
    "name": "Near Doc Suoi Roundabout",
    "address": "Trieu Tien Village, Son Nam Ward, Hung Yen Province",
    "maps_embed_url": "YOUR_GOOGLE_MAPS_EMBED_URL",
    "maps_directions_url": "YOUR_GOOGLE_MAPS_LINK"
  },
  "couple": {
    "groom_name": "Luu Hoang Kien",
    "bride_name": "Pham Thi Hang",
    "title": "Hoang Kien & Pham Hang"
  }

```
